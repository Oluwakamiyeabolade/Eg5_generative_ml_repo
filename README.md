## Overview

The kinesin spindle protein **Eg5 (KIF11)** is overexpressed in multiple cancer types and is essential for bipolar mitotic spindle assembly, making it a validated anti-cancer drug target. This project applies a **generative AI + virtual screening** strategy to identify novel Eg5 inhibitors:

1. **Generative LSTM model** pretrained on 1.8M ChEMBL SMILES and fine-tuned on known Eg5 inhibitors to generate novel candidate molecules
2. **MPNN-based bioactivity prediction** (ChemProp) to prioritize generated compounds
3. **PAINS filtering** to remove pan-assay interference compounds before docking
4. **t-SNE visualization** of chemical space coverage (prior vs. fine-tuned model output)
5. **EDA and molecular property analysis** of generated compounds
6. **Molecular docking & MD simulation** for top hits (described in the paper)

**Compound 103** emerged as the top hit — a bioisostere of the reference inhibitor STLC — and showed superior binding affinity and MD stability compared to the reference ligand.

## Methodology

### 1. Generative Modeling (see paper)
- An **LSTM** language model was pretrained on 1.8M standardized SMILES from ChEMBL
- Transfer learning / fine-tuning was performed using SMILES of known Eg5 inhibitors
- The fine-tuned model generated a library of novel candidate molecules

### 2. Bioactivity Prediction (see paper)
- A **Message Passing Neural Network (MPNN)** via ChemProp was trained to predict Eg5 pIC50
- The model was applied to score the generated library and produce `df_to_dock.csv`

### 3. PAINS Filtering (`EG5_before_docking.ipynb`)
- Candidate molecules were screened against PAINS substructures using:
  - RDKit's built-in `FilterCatalog` (PAINS catalog)
  - A custom `pains_list.csv` SMARTS reference
- Molecules matching PAINS patterns were flagged and visualized with highlighted substructures
- Clean compounds were exported to `results/final_to_dock_filtered_pains.csv`

### 4. Chemical Space Visualization (`EG5_visualize.ipynb`)
- **Morgan fingerprints** (radius=2, 2048 bits) were computed for all compounds
- **t-SNE** dimensionality reduction was applied for 2D projection
- Interactive **Plotly** scatter plots compare:
  - Prior model output vs. Fine-tuned model output
  - Fine-tuned model output vs. Known Eg5 inhibitors (ChEMBL)

### 5. Exploratory Data Analysis (`EG5_visualize.ipynb`)
- Full RDKit descriptor set calculated for all compounds
- **SA Score** (synthetic accessibility) computed via RDKit Contrib
- Distribution plots for structural descriptors (ring counts, H-bond donors/acceptors, etc.)
- Scatter plots of physicochemical properties: MW, LogP, QED, TPSA, FractionCSP3

## Usage

Run the notebooks in order:

```bash
# 1. Filter candidates before docking
jupyter notebook notebooks/EG5_before_docking.ipynb

# 2. Visualize chemical space and top hits
jupyter notebook notebooks/EG5_visualize.ipynb
```

Ensure the required data files are in `data/` before running (see `data/README.md`).

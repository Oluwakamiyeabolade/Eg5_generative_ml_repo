# Eg5_generative_ml_repo

# Data Files

## Input Data

### `data_ML.csv` — Known Eg5 Inhibitors (ChEMBL)
Reference library of experimentally validated Eg5 inhibitors sourced from ChEMBL.  
Required columns: `Molecule ChEMBL ID`, `Smiles`  
Download from: https://www.ebi.ac.uk/chembl/ (search target: Eg5 / KIF11)

### `1m_generated.csv` — Prior Model Output
SMILES generated by the pretrained (prior) LSTM model before fine-tuning.  
Column: `0` (SMILES strings)  
Not included due to file size — generated by running the LSTM prior model described in the paper.

### `unique.csv` — Fine-tuned Model Output
Deduplicated SMILES generated by the fine-tuned LSTM model.  
Columns: `0` (SMILES), `label`  
Not included due to file size — generated by running the fine-tuned LSTM model described in the paper.

### `df_to_dock.csv` — ChemProp-Scored Candidates
Candidate molecules shortlisted by the MPNN (ChemProp) bioactivity model, passed as input to PAINS filtering.  
Column: `Smiles`  
Generated by running the ChemProp prediction pipeline on the fine-tuned model output.

### `pains_list.csv` — PAINS Substructure Reference
Space-separated file of PAINS substructure SMARTS patterns.  
Columns: `smarts`, `name`  
A standard reference list; can be obtained from: https://github.com/rdkit/rdkit or published PAINS literature.

---

## Output Data

### `../results/final_to_dock_filtered_pains.csv`
PAINS-filtered compound list exported after running `EG5_before_docking.ipynb`.  
These compounds were passed to molecular docking in AutoDock Vina / Glide (see paper).
